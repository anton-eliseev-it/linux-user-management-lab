# Linux User Management Lab

This project demonstrates basic Linux user, group, and permission management
in a multi-user environment, following least-privilege principles.

## What was done
- Created users and groups for different roles
- Configured file and directory permissions
- Applied least-privilege access model
- Verified access using different user accounts

## Skills practiced
- Linux user and group management
- File permissions (chmod, chown)
- Basic system administration
- Troubleshooting access issues
